# Twoslash Test

This page demonstrates TypeScript twoslash functionality in code blocks.

## Basic Type Hints

Hover over variables to see their types:

```ts twoslash
interface Todo {
  title: string
  completed: boolean
}

const todo: Todo = {
  title: 'Learn Loro',
  completed: false
}
// ^?

const upperTitle = todo.title.toUpperCase()
//    ^?
```

## Error Display

Twoslash can show TypeScript errors:

```ts twoslash
// @errors: 2540
interface ReadonlyTodo {
  readonly title: string
  readonly completed: boolean
}

const todo: ReadonlyTodo = {
  title: 'Delete inactive users',
  completed: false
}

todo.title = 'Hello' // This will show an error
```
